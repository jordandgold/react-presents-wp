Getting Started

1. Including External Files

Add the required files in your page.

<!-- Core Skewslider styles --> <link rel="stylesheet" href="css/skw-core.css"> <!-- Default Skewslider theme --> <link rel="stylesheet" href="css/skw-default.css"> <!-- The Plugin --> <script src="js/skewSliderSvg.js"></script>

2. HTML Structure

Add the following markup, each slide needs a skw-slide class and a data-bg attribute.

<div id="slider1"> <div class="skw-slide" data-bg="img/1.jpg"></div> <div class="skw-slide" data-bg="img/2.jpg"></div> <div class="skw-slide" data-bg="img/3.jpg"></div> </div>

3. Initialization

Initialize the slider at the end of the body section. see Options section for slider settings.

<script> var slider1 = new skewSlider('#slider1', { // options go here height: 400, navigationDots: false }); </script>

4. Images

You can add images via data-bg attribute on each slide.

<div id="slider1"> <div class="skw-slide" data-bg="path/to/image 1"></div> <div class="skw-slide" data-bg="path/to/image 2"></div> <div class="skw-slide" data-bg="path/to/image 3"></div> ... </div>

5. Captions

To add captions put any html inside each slide.

<div id="slider1"> <div class="skw-slide" data-bg="img/1.jpg"> <div> Image Caption 1 </div> </div> <div class="skw-slide" data-bg="img/2.jpg"> <div> Image Caption 2 </div> </div> <div class="skw-slide" data-bg="img/3.jpg"> <div> Image Caption 3 </div> </div> ... </div>

Responsive Features

you can use the breakpoints option to set different configuration options for different screen sizes, you can set as many breakpoints as you want, to create a new breakpoint, create a new object inside the breakpoints option, give it a name, and set the max width breakpoint using the 'maxWidth' option inside it, for example:

breakpoints: { tablet: { maxWidth : 1024, slidePercent : 45, showCaption: false, skew: -15 }, phone: { maxWidth : 600, slidePercent : 65, skew: -12 } }

And thats all, now you can override any option depending on the user viewport width.

CSS Selectors

List of css classes you may want to use for styling or controlling css animations / states

skw-currentSlideBefore

applied to the current slide at the beginning of the animation.

skw-currentSlideAfter

applied to the current slide when the animation is over.

skw-prevSlide

applied to the previous slide.

skw-nextSlide

applied to the next slide.

Plugin Methods

Update

Update the plugin options at any time with animated transitions.

// initialization var slider1 = new skewSlider('#slider2', { height: 400, navigationDots: false }); // basic usage - when used without new options, it will restore the original options. slider1.update(); // setting new options slider1.update({ slidePercent: 75, skew: 0 });

goTo

Navigate to any slide

// parameters model slider1.goTo(item position / direction string, callback); // moving one position forward slider1.goTo('next'); // moving one position backwards slider1.goTo('previous'); // jumping to any position slider1.goTo(4);
